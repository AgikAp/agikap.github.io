Tabsgame Store

Sebuah website untuk melakukan top-up game secara otomatis. Saya membangun website ini mulai dari Frontend hingga Backend menggunakan Framework PHP Codeigniter 4. Menggunakan API Rest sebagai pemrosesan data dan webhook. Website ini sudah terintegrasi dengan payment gateway yang membuat pembayaran lebih mudah. Untuk membangun website ini hingga deploy ke server kurang lebih menghabiskan waktu 2 - 3 bulan. Anda bisa kunjugi website ini di https://tabsgame.store/
